# Encoder/Decoder Toolkit - Release v1.0.0

## 📥 Download

Two installation options are available:

### **Option 1: Installer (Recommended)**
- **File**: `encoder-decoder-app Setup 1.0.0.exe`
- **Size**: ~93 MB
- **Installation**: Run installer and follow prompts
- **Uninstall**: Use Windows Add/Remove Programs
- **Advantage**: Standard Windows installation, automatic updates

### **Option 2: Portable Executable**
- **File**: `encoder-decoder-app 1.0.0.exe`
- **Size**: ~93 MB  
- **Installation**: No installation required, run directly
- **Advantage**: No system modifications, can run from USB drive

## 🚀 Installation & Usage

### Using the Installer
1. Download `encoder-decoder-app Setup 1.0.0.exe`
2. Double-click to run the installer
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut
5. Enjoy!

### Using the Portable Version
1. Download `encoder-decoder-app 1.0.0.exe`
2. Double-click to launch immediately
3. No additional steps needed

## 🎯 Features Highlights

### Encoding Tools
Base64, Base32, Base58, Base85, Base91, URL Encoding, Hexadecimal, UTF-16, UTF-32, and more

### Cipher Tools  
ROT13, ROT47, Caesar, Atbash, Vigenère, Affine, Playfair, Hill Cipher, Polybius Square, and more

### Cryptographic Tools
XOR Cipher, Bit Rotation, MD5, SHA1, SHA256 hashing
