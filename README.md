# 🔐 Encoder/Decoder Toolkit

**A comprehensive desktop application for encoding, decoding, and analyzing various cipher and encoding formats.**

Perfect for CTF (Capture The Flag) competitions, cybersecurity professionals, and cryptography enthusiasts.

TL:DR: This is a vibecoding product, contained 30+ tools for decoding, from text, zip to simple stegnography. Now i just have to ask AI for the type of encoded and decode it myself instead of wasting token and time on waiting for AI to decode it. Feel free to use it or share any issues!

## 📋 Features

### 30+ Built-in Tools

- **Base Encodings**: Base64, Base32, Base58, Base85, Base91, Hex, URL, Binary
- **Unicode**: UTF-16, UTF-32, Punnycode
- **Ciphers**: ROT13, ROT47, Caesar, Atbash, Vigenère, Rail Fence, Baconian
- **Advanced Ciphers**: Affine Cipher, Playfair, Hill Cipher, Polybius Square
- **Cryptographic**: MD5, SHA1, SHA256, XOR, Bit Rotation
- **Analysis**: Frequency Analysis, Substitution Solver, Vigenère Brute Force
- **Specialized**: Steganography, JWT Decoder, Morse Code, JSON Formatter
- **Email**: Quoted-Printable, Uuencode
- **Esoteric**: Brainfuck, Ook! (Esoteric languages)

### Smart Features

- 🔍 **Auto-Detection** - Automatically identifies encoding types with confidence scores
- 🔎 **Real-time Search** - Instantly find tools by name, category, or description
- 🌙 **Dark/Light Mode** - Toggle between themes for comfortable usage
- 📜 **History Tracking** - Access your last 50 operations with one click
- ⚡ **Performance Metrics** - Shows processing time for each operation
- 🔒 **100% Offline** - All processing happens locally, no external connections
- 📋 **Copy to Clipboard** - One-click copy of results

## 🚀 Quick Start

### Download & Install

1. Go to [Releases](https://github.com/nevr-ine/encoder-decoder-app/releases)
2. Download one of:

4. Enjoy!

### System Requirements

- **OS**: Windows 7 or later (64-bit)
- **RAM**: 256 MB minimum (512 MB recommended)
- **Disk Space**: ~500 MB
- **Internet**: Not required

## 💡 How to Use

1. **Select a tool** from the sidebar (or search for it)
2. **Choose operation**: Encode or Decode
3. **Paste/Type** your input text
4. **Click Process** or press Enter
5. **View results** and copy using the copy button

### Auto-Detection

When in **Decode mode** with input longer than 10 characters:
- The app automatically detects potential encoding formats
- Suggestions appear with confidence percentages
- Click any suggestion to auto-apply the decoder
